# 🎵 Emotion Music Recommender

An AI-powered music recommendation system that detects a user's emotion and suggests songs that match their current mood. The project combines Machine Learning, Computer Vision, and a user-friendly web interface to deliver a personalized music listening experience.

---

## 🚀 Features

- 😊 Real-time emotion detection
- 🎼 Personalized music recommendations
- 📷 Face emotion recognition using webcam
- 🎧 Mood-based playlist generation
- 📊 Interactive and responsive web interface
- ⚡ Fast prediction using Deep Learning models
- 🎵 Multiple emotion categories (Happy, Sad, Angry, Neutral, Surprise, Fear)

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/PRASAD1630/Emotion_Music_Recommender.git
```

### Navigate to the project

```bash
cd Emotion_Music_Recommender
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://localhost:5000
```
